# Exceeding Market Returns

## Summary

The core principle of outperformance is that excess returns above market beta require taking on risk. However, a disciplined approach combining correlation management, position sizing, and options-based hedging can theoretically deliver market-beating returns while simultaneously reducing portfolio risk—a seemingly paradoxical outcome that would be universally adopted if reliably executable.

## Key takeaways

- **Risk as the source of alpha** [00:00] — Excess returns above market return require risk-taking; the question is how to take that risk intelligently.
- **Hedging as a risk-reduction tool** [00:00] — Buying options to cap downside while maintaining upside exposure is a core mechanism for managing risk in an outperformance strategy.
- **Multi-factor risk management** [00:00] — Smart approaches include managing correlation, sizing positions appropriately, and layering hedges—not relying on a single lever.
- **The paradox of achievability** [00:31] — If outperformance with lower risk were reliably doable, everyone would already be doing it; the challenge lies in consistent execution.
