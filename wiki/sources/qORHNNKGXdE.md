# Priority list #optionstrading #trading #stockmarket

## Summary

Eric outlines a foundational time-management and focus discipline for new traders: scheduling dedicated work blocks with a single primary objective, capturing peripheral questions for later investigation, and resisting the urge to chase every tangent. This structured approach prevents scattered learning and ensures meaningful progress on core trading competencies.

## Key takeaways

- **Schedule dedicated work blocks with a single primary objective** [00:17] — assign yourself a specific thing to accomplish in each session, rather than attempting multiple topics simultaneously.
- **Capture peripheral questions for separate work blocks** [00:35] — when research raises tangential questions, write them down and create dedicated time to explore them later, rather than derailing the current session.
- **Purposeful structure prevents distraction** [00:35] — the system is intentionally designed to keep you focused on the main goal while allowing flexibility to add secondary topics as they fit.
- **This discipline remains valuable long-term** [00:35] — Eric still applies this method today, indicating it scales beyond the beginner phase.
