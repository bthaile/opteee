# options for leverage

## Summary

Options provide tactical leverage compared to outright stock ownership. A long-dated call position on Tesla with equivalent notional exposure requires significantly less capital upfront but delivers amplified returns on the same percentage move—at the cost of time decay if the underlying stalls.

## Key takeaways

- **Leverage comparison** [00:00–00:26]: Buying 100 shares of Tesla at $317 requires ~$31,000 capital; a 15% move nets ~$4,761. The same 15% move via long-dated calls nets ~$15,800 on the same capital—demonstrating the leverage multiplier.
- **Time decay trade-off** [00:26–00:52]: Leverage cuts both ways. If Tesla doesn't move, extrinsic value erodes. Mitigation: extend duration (buy further out in time) and apply systematic option-buying discipline.
- **Tactical positioning** [00:52]: Options enable precise, capital-efficient positioning beyond simple buy-and-hold equity.
