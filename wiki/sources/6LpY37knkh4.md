# Trading Breakouts with Options

## Summary

This video covers the foundational process for building an options trade around a breakout thesis. The host emphasizes the importance of defining three key elements: the expected direction and magnitude of price movement, the duration over which that move will occur (modeled via days to expiration), and how to use prior price action as a template for sizing the expected move.

## Key takeaways

- **Thesis generation requires specificity** [00:00]: When building a breakout trade, you must decide where price might go and how it will look as it gets there—not just direction, but magnitude and path.
- **Use prior moves to model expected magnitude** [00:24]: Reference historical price action on the same security to estimate how far and how fast the breakout might extend.
- **Duration is critical to options selection** [00:24]: Options trades must be anchored to a specific DTE (days to expiration); breakout trades typically model moves over 1–2 week timeframes.
- **Expected move sizing** [00:53]: For breakout trades, target expected moves in the 10–12% range, with a realistic upper bound around 10–20%.
