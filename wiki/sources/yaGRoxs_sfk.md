# Basic Steps Before Setting Up a Trade

## Summary
This video outlines the foundational pre-trade checklist: liquidity verification, catalyst identification, and position-sizing alignment with thesis. The host demonstrates a live example where a ~7.8% move over four days is contextualized against a longer-term 12% thesis target, with explicit exit planning around an earnings catalyst on the 16th.

## Key takeaways

- **Liquidity checks are non-negotiable** [00:27] — verify the security can be entered and exited at reasonable spreads before committing capital.
- **Identify catalysts and plan exits around them** [00:27] — the host explicitly plans to exit before the earnings catalyst on the 16th, avoiding event risk.
- **Extend short-term moves into longer-term thesis context** [00:00] — a 7.8% move over four days is evaluated against a 12% longer-term target to assess remaining edge.
- **Complete basic checks before position-sizing** [00:27] — only after liquidity, catalyst, and viability checks should you determine position structure.
