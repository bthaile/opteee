# slugs.md — Canonical Slug Registry

The controlled vocabulary for the OPTEEE wiki. **Every slug used in a page's frontmatter or `[[wikilinks]]` must appear here (or be a registered alias).** `scripts/lint_wiki.py` hard-fails on unregistered slugs — that is the drift guardrail.

To add a slug: add it under the right category in the same change that introduces it. To merge a synonym: add it as an alias (below and via `aliases:` on the canonical page), never create a second page.

Seeded 2026-07-03 from the 14-video prototype. Grows with the corpus.
